# DSA506Hw5_NFLRedZoneEfficiencyDashboard
Streamlit Cloud dashboard analyzing NFL red zone performance

# Dataset information
The dataset used for this dashboard was found on the website NFLSavant.com, which can be found at the following URL: https://nflsavant.com/about.php

The specific dataset used was the 2024 play by play data, which is the pbp-2024.csv file that was downloaded directly from the website at the link above

For future use, for example with the latest 2025 NFL season, one can download that csv file once it is available and replace the 2024 csv
